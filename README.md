# Budget Manager (Console App)

**Budget Manager** is a lightweight, user-friendly Java console application designed to help you manage your personal finances. Track your income, expenses, and balance with ease.

---

## 🚀 Features

- 💰 **Add Transactions**: Record income and expenses effortlessly.
- 📊 **View Balance**: Check your current financial status at a glance.
- 🧾 **Transaction History**: Display a detailed log of all transactions.
- 🔍 **Filter Transactions**: Sort by income or expense for better insights.
- ♻️ **Reset Balance**: Start fresh when needed.
- 🚪 **Exit**: Close the application seamlessly.
- 📁 **Export Data**: Save transaction history to `.txt` and `.csv` files for easy access.

---

## 🛠️ How to Run

Follow these steps to get Budget Manager up and running on your system:

1. **Clone the Repository**  
   Download the project files to your local machine:
   ```bash
   git clone https://github.com/linx7a/budget-manager-java.git
   ```

2. **Navigate to the Project Directory**  
   Move into the project folder:
   ```bash
   cd budget-manager-java
   ```

3. **Compile the Project**  
   Use an IDE or the command line to compile the Java source files:
   ```bash
   javac -d out src/main/java/com/alina/application/Main.java src/main/java/com/alina/domain/*.java
   ```

4. **Run the Application**  
   Launch the app from the command line:
   ```bash
   java -cp out com.alina.application.Main
   ```
